四、论坛报告嘉宾简介

1. 报告人：田聪 教授

题目：大模型时代下的软件形式化方法

摘要：形式化方法一直被视为保障软件可信性与安全性的基石，但其广泛应用长期受制于规约构建和证明过程的高复杂度。随着大语言模型（LLMs）的迅速发展，形式化验证正迎来了新的变革——从需求理解、规约生成到形式化证明，人工智能正以前所未有的方式重塑这一体系。本报告聚焦大模型与形式化方法的深度融合，系统探讨其在三个关键环节的突破：自然语言需求的语义分解与规约生成、多语言形式化推理模型的构建与评测，以及结合静态分析和验证反馈的自动化规约综合。通过研究这些形式化方法在应用时的关键环节，本报告旨在揭示大模型如何成为智能化软件形式化验证的重要支撑力量。

简介：田聪，西安电子科技大学，二级教授，国家级领军人才。长期从事可信软件、形式化验证理论与方法研究，在LICS、CAV、ICSE、FSE、ASE、IJCAI、ISSTA、TOSEM、TSE、软件学报等国内外顶级期刊和会议发表论文200余篇；主持国家自然科学基金重大课题、重点和优青项目、科技部重点研发计划课题、军委装备发展部等项目十余项；担任计算机学会形式化方法专委会副主任，获中创软件人才奖、教育部自然科学一等奖1项、陕西省科学技术一等奖2项；相关研究成果应用于航天控制软件开发和芯片设计EDA软件验证等领域。

2. 报告人：郝丹 教授

3. 报告人：华宇 教授

题目：大内存系统

摘要：大内存系统作为下一代存算架构体系中的重点和核心发展方向，注重将海量数据集完整存储在内存中以突破传统资源墙的限制，为实时智能决策、科学计算等关键负载提供内存级数据访问能力，能够提升数据处理效率与应用响应速度，其核心和关键问题在于如何保障跨节点数据访问的高性能和强一致性。研究工作通过硬件层的缓存一致性协议与软件层的事务内存模型进行深度协同，构建了软硬件协同的高效、可扩展的全局统一内存视图与事务保障机制。

4. 报告人：黄宇 教授

题目：Multi-Grained Specifications for Distributed System Model Checking

摘要：This paper presents our experience specifying and verifying the correctness of ZooKeeper, a complex and evolving distributed coordination system. We use TLA + to model fine-grained behaviors of ZooKeeper and use the TLC model checker to verify its correctness properties; we also check conformance between the model and code. The fundamental challenge is to balance the granularity of specifications and the scalability of model checking.

To address this challenge, we write specifications with different granularities for composable modules, and compose them into mixed-grained specifications based on specific scenarios. For example, to verify code changes, we compose fine-grained specifications of changed modules and coarse-grained specifications that abstract away details of unchanged code with preserved interactions.

We show that writing multi-grained specifications is a viable practice and can cope with model-code gaps without untenable state space, especially for evolving software where changes are typically local and incremental. We detected six severe bugs that violate five types of invariants and verified their code fixes; the fixes have been merged to ZooKeeper. We also improve the protocol design to make it easy to implement correctly.

简介：黄宇，南京大学教授，主要研究领域为分布式算法、分布式系统、形式化验证。在EuroSys、ATC、PODC、TC、TPDS等国际会议与期刊上发表学术论文多篇。主持国家自然科学基金、CCF-华为胡杨林基金、阿里云创新研究计划等多项科研项目。2014年获南京大学登峰人才支持计划资助，协助指导的博士论文获得2016、2017年度中国计算机学会优秀博士论文奖，2022年获SRDS会议最佳论文奖。建设了南京大学“百层次”优质课程《算法设计与分析》，2018年入选首届高校计算机专业优秀教师奖励计划，2024年获评江苏省一流本科课程。

6. 报告人：杨威 副教授

题目：深度学习系统的效率鲁棒性

摘要：深度学习系统已在多种实时应用中得到广泛采用。然而，当这些系统部署在诸如移动设备和物联网设备等资源受限的环境中时，会面临关键的效率挑战。为了缓解计算开销，深度学习系统会动态地根据输入复杂度自适应地调整推理计算。尽管这种动态机制提升了效率，但也引入了新的攻击面——效率对抗攻击（efficiency adversarial attacks），这类攻击会有策略地利用动态机制来降低系统性能。

本文系统性地研究了深度学习系统的效率鲁棒性，提出了首个关于效率攻击的全面分类体系。我们基于三类动态行为对这些攻击进行归类：（i）针对单次推理动态计算的攻击；（ii）针对动态推理迭代的攻击；以及（iii）针对面向下游任务的动态输出生成的攻击。通过深入评估，我们分析了针对深度学习系统效率的对抗策略，并识别出保障系统安全的关键挑战。除此之外，我们还探讨了现有的防御机制，揭示了其局限性，并强调了提升鲁棒性的新需求。研究结果表明，虽然效率攻击日益受到关注，但当前的防御手段仍然不足，亟需创新的缓解策略以保障未来自适应深度学习系统的安全性。

简介：杨威，复旦大学计算与智能创新学院长聘副教授。主要研究方向为先进计算系统，涵盖体系结构设计、系统软件优化以及智能计算平台构建。长期致力于推动高效能计算与智能化应用的深度融合，相关研究成果在前沿计算体系和智能系统领域具有重要学术与应用价值。

8. 报告人：张元良 讲师

题目：Unseen Horizons: Unveiling the Real Capability of LLM Code Generation Beyond the Familiar

摘要：在大模型赋能的代码生成研究中，准确评估其真实能力是推动技术落地的关键前提。然而，现有基准测试存在目标代码泄露、案例时效性差与依赖信息不真实等三大缺陷，导致评估结果难以反映模型在真实、陌生场景下的代码生成能力。本报告介绍一种基于代码混淆的可持续演进的代码生成评估框架，通过符号、结构与语义三个层面对代码与描述进行改写，系统评估大模型的代码生成能力，揭示了现有模型在陌生代码生成任务中的真实局限性。此外，研究还发现即使通过测试的代码仍存在资源管理、性能与鲁棒性等非功能性缺陷，为未来大模型在领域软件中的可靠代码生成与质量保障提供参考。

简介：张元良，国防科技大学计算机学院讲师。2023年博士毕业，2019-2020留学基金委公派至美国伊利诺伊香槟分校联合培养。研究方向为系统软件可靠性和智能化软件工程，主持国家自然科学基金青年项目(C类)/重点项目课题，国防科大青年自主创新基金，在ICSE，FSE，ISSTA，ICLR等会议和期刊发表论文20篇。获得ICSE杰出论文奖（2次）、APSEC杰出论文奖和ICPC杰出论文奖提名，相关工作应用于HDFS、HBase、MySQL等国际知名系统软件。担任系统软件领域国际期刊TOCS, TOSEM, ASEJ的审稿人以及APSEC 2025，SANER 2026程序委员会委员等。

9. 报告人：尹良泽 副教授

题目：Incremental Verification of Concurrent Programs through Refinement Constraint Adaptation

摘要：Programs evolve continuously throughout their life cycles. Verifying each version from scratch is usually impractical, especially for concurrent programs. Designing efficient incremental verification techniques for concurrent programs is highly desired. We focus on the abstraction refinement technique for concurrent program verification. When a program is modified, those refinement constraints generated in the verifications of previous versions are adapted to the new program to avoid redundant analysis. We propose a kernel source based refinement constraint adaptation approach for the scheduling constraint based abstraction refinement method, one of the most efficient abstraction refinement methods for concurrent program verification. Our method supports all kinds of program modifications, and generates adapted refinement constraints according to the modifications. Evaluation on the benchmarks from SV-COMP 2024 and Nidhugg benchmarks shows promising results of our method. Most of the refinement constraints generated in the verification of previous versions can be adapted to the modified program in our experiments. Compared with verifying the modified program from scratch, our incremental verification method can achieve two orders of magnitude speedup for those complex programs.

简介：尹良泽，国防科技大学副教授，主要研究领域为约束求解、模型检验、软件分析与验证等，目前主要研究内容包括基于PThread的并发程序验证、嵌入式中断程序分析验证、操作系统分析验证、Lustre模型检验等，设计开发并行程序验证工具Yogar-CBMC以及嵌入式中断程序分析验证工具，部分成果已在航天、国防等领域中进行应用，研究成果发表在TSE、ICSE、ASE、ISSTA等一系列期刊会议中，并获ICSE 2025杰出论文奖。

10. 报告人：王竟亦 研究员

题目：Formal Methods for Security: From Design, Implementation, to Runtime Monitoring and Beyond

摘要：As modern systems grow in scale and complexity, ensuring security is no longer just about reactive patching — it demands principled guarantees across the entire system lifecycle. In this talk, I will introduce how formal methods may help provide such guarantees by systematically reasoning about security properties from system design, code implementation to runtime. We begin by exploring how formal analysis help catch design flaws early before any code is written, especially in security protocols. We then discuss techniques for verifying security or detecting subtle vulnerabilities in the code, e.g., the OS kernel. Finally, we turn to runtime monitoring, where formal models enable the early detection of security breaches in dynamic and distributed environments. We aim to show that formal methods, when applied judiciously, may serve as a powerful complement to existing engineering workflows — enabling complex systems to be secure by design, by construction, and by observation.

简介：王竟亦，浙江大学控制学院安全所百人计划研究员。新加坡科技设计大学博士，新加坡国立大学博士后。主要研究兴趣是形式化方法与安全，人工智能和软件工程。在S&P、CCS、ICSE、ISSTA、FM、TACAS等国际顶级会议发表论文多篇。研究工作曾两次获得ICSE的杰出论文奖、IEEE TDSC 2024最佳论文奖Runner-Up，并入选ACM SIGSOFT亮点研究。目前担任CCS、ICSE、ASE、ISSTA、WWW等会议PC，ICFEM 2025、大模型安全国际研讨会 2025和SAC-SVT Track 2026 PC Co-chair。主持了国家重点研发子课题、NSFC青年基金C类、浙江省尖兵课题（2项）、CCF-华为胡杨林基金（2项）及蚂蚁、阿里巴巴、华为等产业课题多项。获2024年度中国指控学会科技进步一等奖。

11. 报告人：赵英全 副研究员

题目：程序特征驱动的JVM测试

摘要：Java 虚拟机（JVM）是现代软件生态的核心基础设施之一，其系统结构复杂、功能模块多样，任何潜在缺陷都可能影响整个软件生态的可靠性，因此对其进行系统化的测试至关重要。针对现有方法在测试程序多样性、特征引导及特定模块测试充分性等方面的不足，本报告提出程序特征驱动的JVM测试方法，从语义、结构与行为三层特征出发设计针对JVM的测试方法。研究通过对历史缺陷程序的控制流与数据流特征进行抽象与实例化，提炼可迁移的缺陷诱发模式以提升测试生成的多样性及揭错能力；结合程序特征设计种子程序选择方法，优化输入空间从而提升模糊测试效率；同时设计基于GC事件的测试生成方法与事件覆盖率指标，引导测试对关键模块进行更充分的探索。最终实现从语义抽象到行为感知的程序特征驱动测试方法，为JVM测试提供了新的理论基础与实践路径。

简介：赵英全，天津大学网络安全学院副研究员，2025年6月于天津大学软件工程专业获博士学位，师从陈俊洁教授和王赞教授。主要研究方向为基础软件测试。在相关方向已发表高水平论文14篇，其中CCF-A类论文11篇、中文A类论文3篇。参与重点研发青年科学家项目1项、国自然面上项目1项以及华为横向课题3项。曾获CCF原型竞赛一等奖，入选ChinaSoft 2022优秀博士生论坛等荣誉。

12. 报告人：余航 博士

题目：Code Graph Model (CGM): A Graph-Integrated Large Language Model for Repository-Level Software Engineering Tasks

摘要：Recent advances in Large Language Models (LLMs) have shown promise in function-level code generation, yet repository-level software engineering tasks remain challenging. Current solutions predominantly rely on proprietary LLM agents, which introduce unpredictability and limit accessibility, raising concerns about data privacy and model customization. This paper investigates whether open-source LLMs can effectively address repository-level tasks without requiring agent-based approaches. We demonstrate this is possible by enabling LLMs to comprehend functions and files within codebases through their semantic information and structural dependencies. To this end, we introduce Code Graph Models (CGMs), which integrate repository code graph structures into the LLM's attention mechanism and map node attributes to the LLM's input space using a specialized adapter. When combined with an agentless graph RAG framework, our approach achieves a 43.00% resolution rate on the SWE-bench Lite benchmark using the open-source Qwen2.5-72B model. This performance ranks first among open weight models, second among methods with open-source systems, and eighth overall, surpassing the previous best open-source model-based method by 12.33%.

简介：余航，蚂蚁集团高级算法专家，博士毕业于新加坡南洋理工大学。2020 年加入蚂蚁集团智能平台工程团队，主要利用时序分析、图机器学习、NLP、多模态算法助力技术风险领域智能化。2023年开始从事CodeFuse相关工作。 2024年1月CodeFuse在Big Code Leaderboard上win rate打榜第一，2025年3月CodeFuse-CGM在SWEbench-Lite上issue解决率打榜开源模型第一。同时，在Neurips，ICLR，ICML，Science，T-PAMI等顶会顶刊上共发表论文50余篇。

13. 报告人：章颖强 阿里云资深技术专家

题目：Unlocking the Potential of CXL for Disaggregated Memory in Cloud-Native Databases

摘要：分布式解耦内存池在云原生数据库、AI 推理引擎等场景中变得越来越重要。然而，传统分布式内存架构的软硬件在时延、通信机制和多节点协同方面存在瓶颈。为解决这些挑战，本次报告提出基于 CXL 2.0 交换机 的分离内存系统 PolarCXLMem，并在此基础上设计了即时恢复机制 PolarRecv 与面向多主数据库的缓存一致性协议。该方案在 PolarDB 上完成评估，相较于基于 RDMA 的实现，性能得到显著提升，为 CXL 驱动的分布式解耦内存池在云原生数据库、AI 推理引擎加速等场景中提供了可行的工程路径。

简介：章颖强，阿里云资深技术专家，PolarDB 事务与存储引擎团队负责人，曾多次担任阿里集团“双十一”数据库内核负责人。目前负责阿里云主营战略产品 PolarDB 数据库的内核研发和架构演进。主要研究方向为云原生数据库架构、LLM 推理基础技术等。研究工作获得：中国计算机学会 2024 科技进步一等奖、ACM SIGMOD 2024 工业赛道最佳论文奖、IEEE ICDE 2024 工业赛道最佳论文奖、ACM SIGMOD 2025 工业赛道最佳论文奖等。同时担任 IEEE ICDE 等数据库顶级国际会议的程序委员，其多个技术创新已在阿里云 PolarDB 等产品中广泛应用，并取得显著业务提升。

14. 报告人：张源 教授

题目：访问控制漏洞检测

摘要：访问控制是信息系统防护的基本手段。然而，访问控制规则与业务逻辑高度耦合，导致传统漏洞检测技术失效。由于本次报告将围绕鉴权与授权两大访问控制核心机制，从设计与实现的角度剖析访问控制机制在真实系统中存在的安全隐患，包括鉴权绕过、弱口令、提权、越权等问题。特别地，本次报告将会分享如何利用程序分析、大模型等手段设计自动化的访问控制漏洞检测工具。这些工具在各类真实系统中发现了数百个访问控制漏洞。

简介：张源，复旦大学教授，博士生导师，国家级青年人才，主要研究方向为软件安全，研究工作获USENIX Security 2022杰出论文奖、ACM SIGSOFT 2024杰出论文奖、IEEE S&P 2025杰出论文奖、ACM CCS 2025杰出论文奖，部分研究成果应用于华为、阿里、OPPO、vivo等公司，获华为优秀技术成果奖、阿里巴巴优秀技术合作项目、OPPO产学研优秀合作伙伴、vivo最佳安全技术合作伙伴奖。带领复旦白泽战队获得国内外顶尖安全攻防赛事20余次冠军，发现大量真实软硬件系统高危漏洞，获2021年国家信息安全漏洞库最具价值漏洞奖。
