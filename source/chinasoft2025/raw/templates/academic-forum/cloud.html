云原生与智能化运维

学术论坛 R18

一、论坛介绍

云原生（容器化、微服务、服务网格等）已成为现代应用的基础底座，带来高弹性与快速迭代的同时，也显著提升了运维复杂度。智能运维围绕可观测数据、统计学习/因果推断与优化等方法形成了较为成熟的技术与工程体系。顺应近两年大语言模型（LLM）与智能体（Agent）的突破，我们努力探索以 AI原生（AI-Native）作为方法论与工程原则，将LLM和Agent作为一等公民融入智能运维，增强知识获取、跨域推理与自治执行，使智能运维在既有基础上进一步迈向“意图-到-结果”的智能闭环。与此同时，LLM 作为云端大规模在线服务也需要工程化运营，在确保可靠、可控与合规的同时，实现降本增效、降低时延与规模化扩展。本论坛将邀请学术界与工业界专家分享最新研究与大规模实践，通过主题报告与圆桌讨论，系统探讨方法与系统、工程与治理、评测与实践的协同演进，推动云原生底座上的智能化运维迈向可持续、可度量、可复用的新范式。

四、论坛报告嘉宾简介

1. 报告人：陈鹏飞 教授

题目：面向serverless云原生系统的性能诊断与优化方法研究

摘要：无服务器计算（serverless）因为无需自建服务器、按请求付费，具有极致的计算效用，成为云原生系统的重要计算模式。另一方面，随着以ChatGPT、DeepSeek为代表的大模型的兴起，越来越多的基于大模型的系统开始出现。为了降低成本、提升计算效用，这类软件系统以serverless的方式运行。但是受到运行环境、软件bug的影响，基于serverless的云原生系统也存在诸多性能问题。本次报告围绕基于serverless的云原生系统的性能诊断与优化问题，介绍基于事件图模型的性能诊断方法、基于多智能体强化学习的配置优化方法以及面向大模型系统的性能优化方法。

简介：中山大学计算机学院教授，博士生导师，量子计算与软件研究所副所长，广东省大数据分析与处理重点实验室副主任， 广东省杰出青年基金获得者，中山大学逸仙学者。曾获2020年陕西省计算机学会首届优秀博士论文，ACM FSE 2024(CCF A类会议)杰出论文奖，IEEE ISSRE 2022大会唯一最佳论文。主要方向为：分布式系统、操作系统、计算机网络、软件可靠性等。近年来在国际会议如ASPLOS、FSE、ICSE、ASE、WWW、ACM SoCC、IEEE INFOCOM、IEEE DSN等和期刊如IEEE TON、TPDS、TOSEM、IEEE TNNLS、IEEE TDSC等发表100余篇论文。共承担了30余项项目包括国家重点研发计划课题、国家自然基金面上、青年项目、广东省、广州市科技项目。此外，还与阿里巴巴、华为、腾讯、微信、深信服等企业开展了校企合作项目，并且将部分研究成果在企业落地。

2. 报告人：康昱 微软DKI（数据、知识、智能）团队首席研究员

题目：从可靠性保障到资源优化：智能Agent在云原生运维中的实践

摘要：面对云原生系统复杂性的持续攀升，我们在运维生命周期的关键阶段创新性地运用了不同形态的智能Agent技术。在运行保障阶段，ServiceOdyssey通过自学习Agent实现微服务自治管理，无需服务配置先验知识即可自主适应复杂环境，降低人为操作错误风险。在监控响应阶段，Triangle采用多Agent协作机制，通过语义蒸馏和多角色协商技术，在真实生产环境的数据实验中，达到高事件分类准确率并大大缩短了响应时间。而在资源配置阶段，ProtoRAIL风险感知主动模仿学习Agent，通过从历史数据中学习资源使用模式的原型，实现云环境vCPU资源的自适应超售分配，在保障服务质量的前提下，显著提升资源利用率并大幅降低运营风险。这三项成果系统性地覆盖了云原生运维的核心阶段，从运行期自治管理到故障期快速响应，再到配置期资源优化，形成了完整的智能运维技术体系。通过智能Agent技术的深度应用，我们不仅显著提升了运维效率和系统可靠性，更为云原生运维的智能化转型提供了可复制的最佳实践路径，对推动整个行业的技术演进具有重要参考价值。

简介：微软DKI（数据、知识、智能）团队的首席研究员，并担任复旦大学计算机科学学院行业硕导及香港中文大学名誉副研究员。他长期专注于人工智能、软件工程与系统架构的交叉研究，致力于通过数据驱动方法为智能云服务提供智能化技术。他的研究工作涵盖多个重要方向：在云计算智能化领域，他将人工智能、大数据与云计算相结合，针对云环境运行产生的大规模数据研发先进AI技术，推动大模型智能体的研究和应用，特别是针对Cloud和AIOps等领域，解决关键云服务故障管理问题；在代码智能化领域，他专注于将AI技术应用于复杂工程场景，突破代码行级生成，聚焦仓库级任务如代码库多语言翻译、跨平台迁移和全链路自动化调试。通过与微软Azure、M365、Teams、Copilot等产品团队紧密协作，他将多项创新技术融入核心云平台服务，显著提升了开发效率与系统可靠性。在学术与产业合作方面，他在人工智能、软件工程与系统领域的顶级会议和期刊上发表了多篇论文，与全球顶尖高校及企业保持合作，持续构建其学术与产业影响力。

3. 报告人：王新盟 字节跳动扣子技术负责人

题目：从原型到生产：数据驱动的 AgentOps 如何构建 Agent 效果提升飞轮

摘要：AI Agent 从原型搭建完成到实现生产级可用，需跨越效果验证、风险管控与持续迭代的多重挑战，而 AgentOps 正是应对这一过程的核心实践框架 —— 通过系统化检测、量化评估与动态优化，推动 Agent 效果持续进阶。本文提出一套覆盖 Agent 全生命周期的 AgentOps 解决方案，以 “数据驱动” 为核心支柱，构建 “评测 - 观测 - 优化” 闭环：其一，打造基于 LLM as Judge 的智能化评测体系，实现对 Agent 能力的精准量化与多维度评估；其二，搭建面向 Agent 特性的下一代可观测体系，实时捕捉运行状态、异常风险与用户交互反馈；其三，落地数据驱动的迭代机制，融合 Prompt 工程优化与模型微调能力，形成 Agent 效果提升的增长飞轮，最终确保 Agent 从实验室原型平稳过渡至高可靠、高可用的生产环境。

简介：抖音集团扣子技术负责人，字节跳动AgentOps团队负责人，专注Agent平台建设和研究，在字节内部从零到一孵化了Agent平台，并助力豆包、抖音、飞书、Trae等业务的Agent构建和效果优化。

4. 报告人：裴昶华 副研究员

题目：AgentOps：多智能体系统运维新方向

摘要：Agentic AI系统因灵活性强、搭建门槛低等优势，在科研与产业领域备受关注。然而，当前智能体系统普遍面临稳定性不足、安全性存疑等问题，成为制约其规模化应用的核心瓶颈。在此背景下，一套全面、系统的智能体系统运维方案已成为行业迫切需求，本报告讲重点涵盖AgentOps（Agent System Operations），并对其四大核心阶段（监控、异常检测、根因分析、故障消除）进行定义与可用方法介绍，系统的综述现有多智能体系统可能存在的异常和故障类型，并探讨可落地的技术方向。

简介：裴昶华，中国科学院计算机网络信息中心副研究员，博士生导师，学位委员会委员，中国科学院“百人计划”引进青年人才，从事智能运维、AI for Networking交叉学科研究，相关成果获得MIT Technology Review 和Hacker News报道，在SIGCOMM, ICML, FSE, WWW, SIGKDD, SIGIR, INFOCOM等国际会议上发表论文40余篇。获得2019、2023、2025年最佳论文奖及提名，CIKM 2019论文成为CIKM历史引用量排名前三的论文之一。担任中国计算机学会2024（第七届）CCF国际AIOps挑战赛程序委员会主席。承担和参与中国科学院“十四五”网信专项，国家重点研发计划青年科学家等项目。

5. 报告人：胡昊 华为ICT公共开发部首席资深架构师

题目：提升智能体应用置信度的另一条路径--形式化验证器

摘要：在智能应用快速发展的今天，大语言模型在专业领域的解答置信度仍然是较大的挑战，通过模型微调（无论是sft还是lora）引入领域知识代价仍然昂贵，且效果不稳定；RAG虽有一定帮助，但更像拿着小抄来考试：针对题库中未直接包含的问题，效果并不理想。本主题介绍一种新的路径：借助PLT领域的类型系统建模实现对LLM进行校验、反馈迭代的思路。这种思路使得未微调的通用模型在领域专有问题域上的回答正确率提升近3倍。

简介：华为公司软件领域科学家委员会委员，华为公司 ICT 公共开发部首席资深架构师，openFuyao 社区技术委员会主席。

6. 报告人：王昊奋 特聘研究员

题目：AI原生记忆操作系统MemOS：从上下文工程到记忆工程

摘要：记忆能力是智能体从被动响应迈向主动进化的核心，其分层架构（瞬时→短期→长期）虽受人类认知机制启发，却长期受限于碎片化、不可控、低效迭代等瓶颈。本报告将分享操作系统级记忆框架MemOS如何重构这一范式：通过将记忆升维为可调度的「一级资源」，以MemCube原子单元统一封装明文记忆（结构化知识）、激活记忆（瞬时状态）、参数记忆（固化常识），实现动态协同；突破传统上下文工程局限，依托异步预加载机制实现跨场景记忆预测（首Token响应加速94.2%），在LoCoMo评测中开放域任务准确率提升19.4%，验证记忆工程在金融风控、个性化学习等Knowhow沉淀场景的进化潜力。

简介：同济大学百人计划特聘研究员，博士生导师。他是全球最大的中文开放知识图谱联盟OpenKG轮值主席。他负责主持多项国家AI专项，发表100余篇AI领域高水平论文。他构建了全球首个可交互养成的虚拟偶像—“琥珀·虚颜”；所构建的智能客服机器人已累计服务用户超过10亿人次。目前，他担任中国计算机学会术语工委副主任，自然语言处理专委秘书长，信息系统专委常委，智能机器人专委会执委；中国中文信息学会理事，大模型专委会指导委员会委员，语言与知识计算专委会副秘书长；中国指挥控制学会大模型专委会常务委员；上海市计算机学会自然语言处理专委会副主任等社会职位。
